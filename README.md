# 📊 HR Analytics Dashboard

A professional **HR Analytics Dashboard** built using **Python, Pandas, Matplotlib, and Streamlit**. This project analyzes HR data to provide actionable insights into employee demographics, attrition, salaries, performance, and workforce trends.

---

## 🚀 Features

* 📌 Interactive HR Dashboard
* 📌 KPI Cards

  * Total Employees
  * Average Salary
  * Average Age
  * Attrition Rate
* 📌 Sidebar Filters

  * Department
  * Gender
  * Job Role
* 📌 Employee Data Table
* 📌 Download Processed Dataset
* 📌 Data Visualizations

---

## 📊 Dashboard Charts

* Employees by Department
* Gender Distribution
* Attrition Analysis
* Salary Distribution
* Age Distribution
* Performance Rating
* Job Satisfaction
* Overtime vs Attrition
* Promotion Analysis
